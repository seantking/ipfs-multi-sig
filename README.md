## Summary

This POC demonstrates the user flows of a multi-signature transaction. Read the RFC for further context [here](https://www.notion.so/RFC-IPFS-MultiSig-b23b1e3ac7e04ce4bfb92d8595ec1135).

## Commands

```
yarn deploy
```

```
yarn build
```

```
yarn start
```
